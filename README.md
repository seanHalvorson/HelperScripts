# HelperScripts
## ymlfldr
This is a ChatGPT generated script to take a YAML file and turn it into a folder structure, the yaml file should be similar to what we have below

```
00-09 Administration:
  00 Index:
    - 00.00 Index.yaml
10-19 Personal:
  11 Me: 
    - 11.01 Journals
    - 11.02 Food
    - 11.03 Workouts
```
